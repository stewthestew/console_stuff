# Console Stuff
Convenient re-exports of indicatif, console, dialoguer and clap

# Installation
Just add this to your `Cargo.toml` file
```toml
[dependencies]
console_stuff = "0.2.0"
```
or
```toml
[dependencies]
console_stuff = { git = "https://github.com/stewthestew/console_stuff" }
```
For the latest stuff in a re-export crate
